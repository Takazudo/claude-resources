# css-box-shadow-layered-natural



# Natural Layered Box Shadow

Create realistic shadows using absolutely positioned blurred elements instead of uniform CSS `box-shadow`. Produces a more natural, directional lighting effect.

## Technique

Use extra `<span>` elements with `absolute` positioning, `rounded-[50%]` (elliptical shape), and `blur()` to simulate natural light casting. The parent must have `relative` positioning.

## Variants

### Bottom-only (lit from above) — default

```html
<div class="relative ...">
  <!-- Top: subtle edge line only -->
  <span class="pointer-events-none absolute top-0 left-0 right-0 h-[1px] bg-black/10" />
  <!-- Bottom: layered shadow -->
  <span class="pointer-events-none absolute bottom-0 left-0 right-0 h-[1px] bg-black/25" />
  <span class="pointer-events-none absolute bottom-[-6px] left-[3%] right-[3%] h-[6px] rounded-[50%] bg-black/30 blur-[6px]" />
  <span class="pointer-events-none absolute bottom-[-18px] left-[5%] right-[5%] h-[12px] rounded-[50%] bg-black/20 blur-[16px]" />
  <!-- content -->
</div>
```

### Both sides (floating card)

```html
<div class="relative ...">
  <!-- Top shadow -->
  <span class="pointer-events-none absolute top-0 left-0 right-0 h-[1px] bg-black/15" />
  <span class="pointer-events-none absolute top-[-12px] left-[3%] right-[3%] h-[4px] rounded-[50%] bg-black/20 blur-[8px]" />
  <!-- Bottom shadow (stronger) -->
  <span class="pointer-events-none absolute bottom-0 left-0 right-0 h-[1px] bg-black/25" />
  <span class="pointer-events-none absolute bottom-[-6px] left-[3%] right-[3%] h-[6px] rounded-[50%] bg-black/30 blur-[6px]" />
  <span class="pointer-events-none absolute bottom-[-18px] left-[5%] right-[5%] h-[12px] rounded-[50%] bg-black/20 blur-[16px]" />
  <!-- content -->
</div>
```

### Top-only (lit from below)

```html
<div class="relative ...">
  <span class="pointer-events-none absolute top-0 left-0 right-0 h-[1px] bg-black/25" />
  <span class="pointer-events-none absolute top-[-6px] left-[3%] right-[3%] h-[6px] rounded-[50%] bg-black/30 blur-[6px]" />
  <span class="pointer-events-none absolute top-[-18px] left-[5%] right-[5%] h-[12px] rounded-[50%] bg-black/20 blur-[16px]" />
  <!-- Bottom: subtle edge line only -->
  <span class="pointer-events-none absolute bottom-0 left-0 right-0 h-[1px] bg-black/10" />
  <!-- content -->
</div>
```

## Tuning

| Parameter | Effect | Range |
| --- | --- | --- |
| `bg-black/N` opacity | Shadow darkness | 10-40 |
| `blur-[Npx]` | Softness | 4px-20px |
| `h-[Npx]` | Shadow thickness | 4px-14px |
| `left/right [N%]` | Shadow narrower than box | 2%-10% |
| Offset (`top-[-Npx]`) | Distance from edge | 4px-20px |

Two blur layers (tight + wide) create depth. The tight layer (smaller blur, closer offset) gives definition. The wide layer (larger blur, farther offset) gives ambient softness.

## Key Rules

- Parent container must have `relative`
- All shadow spans need `pointer-events-none` and `absolute`
- Use `rounded-[50%]` for the elliptical blur shape (not `rounded-full`)
- The `overflow-hidden` on parent will clip the shadows — avoid it or use a wrapper
