# Agents

Global Claude Code subagents reference.

These agents are available globally from `~/.claude/agents/`.

## Available Agents (9)

- [`code-reviewer`](./code-reviewer.mdx) (opus) - code reviewer
- [`markdown-generator`](./markdown-generator.mdx) (opus) - markdown file generator for creating markdown formatted text files based on user-provided context. It handles file management, URL formatting, and follows strict guidelines for markdown generation.
- [`markdown-writer`](./markdown-writer.mdx) (opus) - markdown writer
- [`researcher`](./researcher.mdx) (opus) - researcher
- [`safe-developer`](./safe-developer.mdx) (opus) - standard developer agent for implementing code changes based on user requirements. It follows a strict development workflow with mandatory approval gates and prioritizes safety, correctness, and clear communication.
- [`tdd-developer`](./tdd-developer.mdx) (opus) - TDD developer agent for implementing code changes based on user requirements. It follows a strict development workflow with mandatory approval gates and prioritizes safety, correctness, and clear communication.
- [`text-fixer`](./text-fixer.mdx) (opus) - text fix agent.
- [`zudocg-writer`](./zudocg-writer.mdx) (opus) - Write articles for CodeGrid at the takazudo-codegrid-writing repository. Receives a detailed writing brief and produces a complete article following CodeGrid conventions.
- [`zudoesa-writer`](./zudoesa-writer.mdx) (opus) - Write articles for esa (internal blog) at the takazudo-esa-writing repository. Receives a detailed writing brief and produces a complete article following esa conventions.
