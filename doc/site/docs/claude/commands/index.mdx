# Commands

Global Claude Code custom commands reference.

These commands are available globally from `~/.claude/commands/`.

## Available Commands (5)

- [`/bambulab-height-calculation`](./bambulab-height-calculation.mdx) - Calculate Z coordinates for stacking/merging objects in Bambu Studio.
- [`/dummy-command`](./dummy-command.mdx) - A dummy command for debugging the doc site. Does nothing useful.
- [`/dump-fullpath`](./dump-fullpath.mdx) - Output the full absolute path of a file mentioned in the previous message.
- [`/next-todo`](./next-todo.mdx) - Queue the next task to do after the current one finishes
- [`/start-wt-dev`](./start-wt-dev.mdx) - Prepare a git worktree for development on a GitHub issue.
